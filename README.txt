InfoTrackTechnicalTest

Design:

The design of the project is using .NET Core 6 and Vuejs 3 with Typescript.

API Setup:
We need to migrate the datbase using code first. So open the solution and the "Package Manager console". Change directory to "InfoTrackWebScrapper" by executing commands: cd InfoTrackWebScrapper. Then dotnet database update to update the database to SQL Server.

Web APP Setup:
We are using Vue for the SPA. Need to install vuejs, npm install -g vue/cli. Then execute npm install to install all the project dependencies and execute npm run serve.
